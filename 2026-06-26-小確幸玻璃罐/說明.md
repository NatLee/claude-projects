# 小確幸玻璃罐 · Jar of Small Joys 🫙

**主題類別**：生活痛點小工具（情緒・生活記錄）
**產出形式**：B 類 — 完整小專案（前端 + Python Flask 後端 + SQLite 資料庫）
**日期**：2026-06-26

---

## 這是什麼

一個會「慢慢裝滿」的數位玻璃罐。每天遇到一件微小但確切的好事，就寫一張小紙條投進去——
午後剛剛好的咖啡、路邊願意給摸的貓、終於做完拖很久的事……

紙條會依心情用不同顏色，一張張落進罐子裡堆起來。日子久了，你會擁有一整罐專屬於自己的快樂。
心情低落時，按一下「搖一搖」，罐子會隨機抖出一則過去的小確幸，提醒你其實擁有的不少。

> 「小確幸」一詞源自村上春樹，指生活中微小而確切的幸福。每天記一則，是溫柔對待自己的小練習。

## 怎麼玩

1. 在右側欄寫下今天的一則小確幸（最多 200 字）。
2. 選一個當下的心情：☀️ 溫暖、🌿 平靜、✨ 小驚喜、🍀 幸運、🌟 有成就。
3. 按「投進罐子」，紙條就會掉進左邊的玻璃罐。
4. 想回味時，點罐子或按「搖一搖」，隨機抽出一則。
5. 下方「罐子裡的回憶」可看全部紀錄，按 × 可拿掉某一則。
6. 「罐子狀態」會顯示累積總數、本月收集、連續天數與最常出現的心情。

小技巧：在輸入框按 `Ctrl + Enter` 可快速投入。所有資料存在本機的 `jar.db`，只屬於你自己。

## 怎麼啟動（Windows）

電腦已安裝 miniconda 與 Node.js，後端用 Python（Flask）。任選一種：

**方法 A：一鍵啟動（最簡單）**
直接雙擊資料夾裡的 `run.bat`，它會自動建立虛擬環境、安裝套件並啟動伺服器。
看到畫面提示後，用瀏覽器打開 👉 **http://localhost:5000**
要關閉伺服器：回到黑色視窗按 `Ctrl + C`。

**方法 B：用 conda 手動啟動**
```bat
cd /d "%USERPROFILE%\Desktop\claude-projects\每日小專案\2026-06-26-小確幸玻璃罐"
conda create -y -n joyjar python=3.11
conda activate joyjar
pip install -r requirements.txt
python app.py
```
然後打開 http://localhost:5000

**方法 C：用 venv 手動啟動**
```bat
cd /d "%USERPROFILE%\Desktop\claude-projects\每日小專案\2026-06-26-小確幸玻璃罐"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

> 資料庫 `jar.db` 會在第一次啟動時自動建立，無需手動設定。

## API 一覽

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET | `/` | 主頁面 |
| GET | `/api/joys` | 取得所有小確幸（新到舊） |
| POST | `/api/joys` | 新增一則（JSON：`content`、`mood`） |
| GET | `/api/joys/random` | 隨機回味一則 |
| DELETE | `/api/joys/<id>` | 刪除某一則 |
| GET | `/api/stats` | 統計：總數、本月、連續天數、最常心情 |

## 為什麼有趣 / 實用

把「記錄好事」這件抽象的事，變成看得見、會堆疊、會裝滿的玻璃罐。
每投一則就有紙條落下的回饋，累積感很真實；心情差時搖一搖隨機回味，是一個小小的情緒急救包。
這也呼應正向心理學常提到的做法：刻意留意並記下生活中的小小好事，有助於把注意力放回美好的事物上。
