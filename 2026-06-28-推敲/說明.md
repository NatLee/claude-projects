# 推敲 · 每日成語猜謎 🖌️

> 鳥宿池邊樹，僧敲月下門。

一款水墨宣紙風的「成語版 Wordle」。每天會有一則四字成語當作謎底，你有 **六次** 機會推敲出來。每猜一次，每個字會回饋顏色提示，幫你一步步逼近正解。

這是一個 **純靜態網頁小專案**：出題、判定與計分全部在瀏覽器端以 JavaScript 完成，不需要伺服器、不需要安裝任何東西，**離線也能玩**。

---

## 怎麼玩

1. 在輸入框打出一個 **四字成語**（用你的中文輸入法），按「送出」或 Enter。
2. 每個字會翻牌並上色：
   - 🟩 **綠**：這個字、這個位置都對。
   - 🟨 **黃**：成語裡有這個字，但位置不對。
   - ⬜ **灰**：成語裡沒有這個字。
3. 六次內猜中就贏了！想不出來可以按「給個提示」，或按「看答案」直接揭曉。
4. 「今日一題」全天同一題（依台灣日期決定，同一天所有人看到同一題）；想多玩可切到「練習模式」隨機出題。
5. 按「統計」可看歷來勝率與「幾次猜中」的分佈長條圖。

> 小提醒：「今日一題」完成後會記錄成績、隔天才換新題；當天重開頁面會還原你已完成的盤面。想再過癮就玩練習模式吧。

---

## 如何開啟

**直接用瀏覽器開啟 `index.html` 即可**（雙擊檔案就行），離線可用，不需安裝 Python、也不需啟動伺服器。

你的作答紀錄與統計資料儲存在瀏覽器的 **localStorage**（key 以 `tuiqiao.` 開頭）；清除瀏覽資料（網站資料／localStorage）會一併清除成績。

## 為什麼好玩

- **每天一則、全憑推理**：用排除法與位置線索一步步逼近，像在腦中下圍棋。
- **順便練成語**：猜中或揭曉時會附上成語釋義，玩著玩著就記住了。
- **輕巧離線**：純 HTML/CSS/JS，打開就能玩，成績存在自己的瀏覽器裡。
- **有成就感**：勝率與猜中次數分佈讓你看見自己愈推愈準。

## 關於「推敲」典故

唐代詩人賈島赴京趕考，在驢背上吟得名句「鳥宿池邊樹，僧**敲**月下門」。他在「推」與「敲」二字間反覆斟酌，竟不覺闖入京兆尹韓愈的儀仗。韓愈不但沒怪罪，反而停下與他細究，認為月夜訪門用「敲」字更有聲響與意境。後人遂以「推敲」一詞，形容對文字反覆琢磨、精心斟酌——正是這個小遊戲的精神。

**資料來源**：典出《唐詩紀事》；典故細節經網路資料查證（參見 epochtimes.com、secretchina.com 等對「推敲」典故的記述）。成語釋義為常見通用解釋。
