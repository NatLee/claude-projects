# 你的耳朵幾歲？· 高頻聽力測試 🎧

**主題類別**：網路趣聞・冷知識
**產出形式**：A 精緻互動頁（HTML / CSS / JS，純前端、離線可用）
**日期**：2026-06-29

## 這是什麼

一個用瀏覽器就能玩的「高頻聽力小測試」。它用 Web Audio 即時合成純音，從 8 kHz 開始慢慢往上掃，找出你還聽得見的**最高頻率**，再換算成一個好玩的「耳朵年齡」。

人的高頻聽力會隨年齡悄悄退化——這是真實存在的生理現象，醫學上叫**老年性聽損（presbycusis）**。網路上著名的「蚊子鈴聲」就是利用這點：約 17.4 kHz 的聲音，主要只有 25 歲以下的人受得了。這個頁面讓你親耳驗證自己落在哪一段。

## 怎麼玩

直接用瀏覽器打開 `index.html` 即可（離線也能跑，不需網路）。

1. **戴上耳機**，把音量從小慢慢轉大，找個安靜的環境。
2. 按 **「開始自動掃頻」**，聲音會從 8 kHz 一路往上爬。
3. 當你**再也聽不到**聲音的那一刻，立刻按 **「我聽不到了！」**。
4. 頁面會顯示你聽得見的最高頻率，以及對應的耳朵年齡估算與解說。
5. 想針對特定頻率測試，可展開 **「逐音檢查站」**，分別播放 8 k～20 kHz 的純音（含 17.4 kHz 蚊子鈴聲），每格標著「通常幾歲以下聽得到」。

你的最佳紀錄會用 `localStorage` 存在本機，下次打開還在。

## 為什麼有趣

把「聽力會隨年齡退化」這種抽象知識，變成你**親耳**就能驗證的體驗。很多人第一次發現自己已經聽不到 17 kHz 時都會愣一下——原來歲月連這種地方都留下了痕跡。拿給不同年齡的家人一起玩，差異特別明顯。

## 背後的原理

- 年輕健康的耳朵聽得到約 **20 Hz – 20 kHz**。
- 耳蝸基底部負責高頻的毛細胞最早磨損，所以聽力是「由高音往低音」逐漸流失，日常人聲（約 0.5–4 kHz）反而保留較久。
- 常見的頻率—年齡對照（近似值，個體差異很大）：
  - 17.4 kHz：約 24–25 歲以下（蚊子鈴聲）
  - 16 kHz：約 30 歲以下（過了 25 歲多半就聽不到 16 kHz 以上）
  - 15 kHz：約 40 歲以下
  - 14 kHz：約 50 歲
  - 12 kHz：約 55–60 歲
  - 10 kHz：約 65 歲
  - 8 kHz：幾乎人人都聽得到

## ⚠️ 重要免責聲明

這只是個**好玩的估算**，**不能取代專業聽力檢查**。結果受耳機／喇叭品質、音量、瀏覽器與環境噪音影響極大——一般筆電與手機喇叭往往根本發不出 16 kHz 以上的聲音，聽不到很正常，**不代表你的聽力有問題**。若你對自己的聽力有任何疑慮，請諮詢耳鼻喉科醫師或聽力師。

## 技術小檔案

- 純前端，無任何外部相依套件；以 Web Audio API 的 `OscillatorNode` 即時合成正弦純音。
- 音量保守（gain ≈ 0.12）並加上淡入淡出，避免高頻爆音。
- 畫面波形以 `<canvas>` 即時繪製：音越高，波形越密。
- 切到背景分頁會自動靜音。
- 已在沙箱以 jsdom 模擬載入並通過 8 項互動測試（無主控台錯誤）。
