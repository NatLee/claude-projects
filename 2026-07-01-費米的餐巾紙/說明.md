# 費米的餐巾紙 · 數感練習場 🧾✏️

> 「給我一張餐巾紙，我就能估出整個世界的大小。」

一個訓練「**數量級直覺**」的小遊戲。系統會丟出各種關於世界的問題——從一年有幾秒、人體有幾個細胞，到 1 公升水裡有幾個水分子——你不用算到精確，只要用**係數 × 10 的次方**把答案的「數量級」抓出來。只要落在正解的 **10 倍以內**，就算命中！

這是實用又好玩的思考練習：物理學家費米（Enrico Fermi）就靠這種「拆解 → 估算 → 連乘」的心算，在原子彈試爆時用撒紙片估出了爆炸當量。

---

## 這是什麼類型的專案

- **類別**：學習新知
- **產出形式**：B — 完整小專案（Python Flask 後端 + SQLite 資料庫 + 前端）
- 後端負責：出題、依「差幾個數量級」評分、把你每次的估算存進 SQLite，並統計你的**數感檔案**（命中率、平均誤差、連勝紀錄）。

## 怎麼玩

1. 讀題目（例如：「一個成年人的身體大約由多少個細胞組成？」）。
2. 用兩個轉盤湊出你的估計：
   - **係數**（1～9.9）
   - **數量級**滑桿（10⁰ ~ 10²⁶）——這是重點，先想「大概是 10 的幾次方」。
3. 按「**揭曉答案**」。畫面會用一條**對數刻度軸**把「你」和「正解」兩支旗子擺出來，讓你一眼看出差了幾個數量級，並附上**解題思路**與資料來源。
4. 每一題都會更新下方的「**你的數感檔案**」。目標是把命中數量級率練上去！

> 小提醒：費米估算的金標準是「**落在一個數量級內**」。差 3 倍在這裡也算漂亮，別追求剛好。

## 在 Windows 上啟動

### 方法一：一鍵啟動（最簡單）

直接雙擊資料夾裡的 **`run.bat`**。它會自動建立虛擬環境、安裝 Flask、啟動伺服器。看到提示後，用瀏覽器打開：

```
http://localhost:5000
```

結束請在視窗按 `Ctrl + C`。

### 方法二：用 conda（你已安裝 miniconda）

在此資料夾開啟終端機（Anaconda Prompt），依序執行：

```bat
conda create -n fermi python=3.11 -y
conda activate fermi
pip install -r requirements.txt
python app.py
```

然後打開 http://localhost:5000 。

### 方法三：用內建 venv

```bat
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

> 資料庫 `fermi.db` 會在**第一次啟動時自動建立**並灌入題庫，不需手動處理。
> 若你的桌面資料夾是 OneDrive 同步夾、遇到「database is locked / disk I/O error」，
> 可先設定環境變數再啟動：`set FERMI_DB=C:\Temp\fermi.db`（把資料庫放到本機路徑）。

## API 一覽

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET  | `/api/question?exclude=<id>` | 隨機出一題（不含答案） |
| POST | `/api/guess` `{id, guess}` | 回傳正解、差幾個數量級、評語、解題思路 |
| GET  | `/api/stats` | 你的數感檔案（命中率、平均誤差、連勝） |
| POST | `/api/reset` | 清除所有估算紀錄 |

## 為什麼有趣 / 有用

- **鍛鍊真本事**：數量級估算是工程、科學、面試、日常決策都用得到的思考力。看到一個宣稱的數字，能不能馬上判斷「這合理嗎？」靠的就是它。
- **即時回饋**：對數刻度軸把抽象的「差幾個數量級」畫成看得見的距離，錯得離譜也一目了然。
- **會進步**：SQLite 記錄讓你看到命中率隨練習上升，還有連勝可以拚。
