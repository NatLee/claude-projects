# 冷知識扭蛋機 🥚✨

投一枚硬幣、轉一顆扭蛋，掉出一則「經過查證」的冷知識；抽到的卡片會收進你的專屬圖鑑，慢慢把 24 張全部集滿。這是一個 **前端 + Python（Flask）後端 + SQLite** 的小專案，你的收藏會存進本機資料庫，關掉再打開也還在。

> 類別：網路趣聞・冷知識　·　產出形式：B 完整小專案（Flask + SQLite）

---

## 這是什麼、怎麼玩

- 按下大大的「**轉蛋！**」按鈕，扭蛋機會轉一下、掉出一顆扭蛋，翻開就是今天的冷知識。
- 每張卡片有 **類別**（動物／太空／人體／歷史／食物／自然）與 **稀有度**：
  - 普通　·　稀有 ✦　·　傳說 ★
  - 越稀有的知識越難抽到（權重 6 : 3 : 1），抽到 **傳說** 級還會有金光與 NEW 提示。
- 下方是「**冷知識圖鑑**」：已收集的卡片會亮起來，點一下可以再讀一次；還沒抽到的是灰色問號。
- 上方即時顯示 **已收集 X/24、完成度、轉蛋次數**。集滿 24 張就通關啦！
- 想重來可按最下方「清空收藏，重新收集」。

全部 24 張卡片、6 大類各 4 張，內容皆為真實且查證過的冷知識。

---

## 怎麼啟動（Windows）

這個專案需要用 Python 跑起後端。你的電腦已裝好 miniconda 與 Node.js，二選一即可。

### 方法 A：一鍵啟動（最簡單）
直接雙擊資料夾裡的 **`run.bat`**，它會自動建立環境、安裝套件並啟動伺服器。
看到畫面出現網址後，打開瀏覽器前往：

```
http://localhost:5000
```

要停止伺服器，回到黑色視窗按 `Ctrl + C`。

### 方法 B：用 conda 手動啟動
打開「Anaconda Prompt」，依序輸入：

```bat
cd /d "C:\Users\NatLee\Desktop\claude-projects\每日小專案\2026-07-03-冷知識扭蛋機"
conda create -y -n gacha python=3.11
conda activate gacha
pip install -r requirements.txt
python app.py
```

然後打開瀏覽器前往 `http://localhost:5000`。

### 方法 C：用內建 venv
```bat
cd /d "C:\Users\NatLee\Desktop\claude-projects\每日小專案\2026-07-03-冷知識扭蛋機"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

> 第一次執行時，程式會自動建立資料庫 `gacha.db` 並填入 24 張卡片；若資料庫檔曾損毀，也會自動重建，確保一定跑得起來。你的收藏就存在這個檔案裡。

## 專案結構

```
2026-07-03-冷知識扭蛋機/
├─ app.py            # Flask 後端：轉蛋 / 圖鑑 / 統計 / 重置 API，首次執行自動建 DB
├─ requirements.txt  # 相依套件（只有 Flask）
├─ run.bat           # Windows 一鍵啟動
├─ templates/
│   └─ index.html    # 前端：扭蛋機、翻牌動畫、圖鑑（無外部連線，離線可跑）
└─ 說明.md
```

主要 API：`POST /api/pull`（轉蛋）、`GET /api/album`（整本圖鑑＋統計）、`POST /api/reset`（清空收藏）。
