# 深夜電台 · Lo-Fi 節拍工坊 🎛️

一台開在瀏覽器裡的迷你節拍機。點亮格子，就能敲出一段慵懶的 Lo-Fi 節拍——而且**所有聲音都是即時合成的，沒有用任何取樣檔案**，離線也能玩。

> 類別：創意・娛樂　·　形式：C 自包含互動單頁（純 HTML/CSS/JS，零相依）

---

## 這是什麼

一個 16 步的「步進式序列器」（step sequencer）。畫面上是一張 6×16 的格子表：

| 音軌 | 是什麼 |
|---|---|
| 大鼓 | 正弦波振盪器 → 音高俯衝 → 低通濾波，厚實的 Boom |
| 小鼓 | 白噪音 + 三角波音體，清脆的 Pah |
| 腳踏鈸 | 高通白噪音，短促的 Tss（每次力度略有隨機，聽起來更像人手） |
| 拍手 | 四層錯開的噪音爆點，模擬一群人拍手 |
| 貝斯 | 前半小節 A、後半小節 F，低頻正弦墊底 |
| 電鋼琴 | A 小調五聲音階，依欄位上行——**怎麼點都不會走音** |

橫向 16 格代表一小節裡的 16 個十六分音符，播放頭會由左往右掃過，掃到亮著的格子就發聲、循環播放。

## 怎麼玩

1. 用瀏覽器打開 `index.html`（Chrome / Edge / Safari 皆可）。
2. 直接**點格子**開關聲音，點下去當下就會試聽那一聲。
3. 按大大的 **▶** 或鍵盤 **空白鍵** 開始／停止循環。
4. 三個旋鈕：
   - **速度 BPM**：60–140，慢一點更 chill。
   - **搖擺 Swing**：把奇數拍往後挪一點點，節奏就「鬆」下來了，這是 Lo-Fi 的靈魂。
   - **音量**。
5. 兩顆環境音開關：**◐ 黑膠**（黑膠唱片的細碎雜訊）與 **╱ 雨聲**，疊上去氛圍瞬間到位。
6. 四種預設起手式：**慵懶 Lo-Fi / 都會 City / 雨夜 Rainy / 頓拍 Boom-Bap**；或按 **🎲 隨機生成** 骰出一段全新的。
7. 你的節拍、速度、開關、濾波位置都會自動存在這台裝置（localStorage），**明晚打開它還在**。

畫面上的磁帶捲盤會跟著拍子轉、右邊的 VU 表會隨大鼓小鼓跳動、玻璃窗上還下著雨——純粹為了氣氛。

## DJ 音效台（新）

控制盤下半排是給你「玩現場」用的，全部即時合成、可搭配鍵盤快捷鍵：

- **FILTER 濾波旋鈕**：上下拖曳掃頻——往上轉切掉低頻變單薄（高通）、往下轉悶起來（低通），中間通透；**雙擊回中**。這是 DJ 最經典的一招，播放中掃一下張力就來了。
- **音效墊**（按鈕或數字鍵）：
  - <kbd>1</kbd> **氣笛**：疊三顆鋸齒波加抖音的經典 reggae/派對氣笛。
  - <kbd>2</kbd> **刮碟**：來回變速的黑膠刮盤聲。
  - <kbd>3</kbd> **上升**：頻率一路爬升的 riser，副歌前拉緊情緒。
  - <kbd>4</kbd> **打擊**：低頻 boom＋噪音撞擊，重拍砸下去。
  - <kbd>5</kbd> **雷射**：復古 pew 音。
- **回音**（按住 <kbd>E</kbd>）：按住把當下的節拍甩進節奏延遲，放開讓尾巴自然殘響開來。
- **煞車**（<kbd>B</kbd>）：唱盤煞停——音高一路滑落並停止播放，像把手按在黑膠上。

快捷鍵一覽：<kbd>空白鍵</kbd> 播放／停止、<kbd>1</kbd>~<kbd>5</kbd> 音效、<kbd>E</kbd> 回音、<kbd>B</kbd> 煞車。

## 為什麼有趣

- **一秒上手，很難做難聽**：旋律軌走五聲音階、鼓點對齊格線，隨便點都成調。門檻低、上限高。
- **看得見的合成音**：這頁最好玩的地方是——它**沒有音效檔**。每一聲大鼓、小鼓、貝斯，都是在你按下播放的瞬間，用 Web Audio 的振盪器與濾波器「算」出來的。
- **一個小小的致敬**：1980 年 Roland 推出傳奇鼓機 **TR-808**（TR = _Transistor Rhythm_）。它跟同期對手最大的不同，是用**類比合成**產生聲音，而不是播放取樣；它那顆著名的大鼓，就是「正弦波振盪器 + 低通濾波器 + 放大器」做出來的。TR-808 一開始因為聲音「不夠真實」而滯銷，全球只生產約 1.2 萬台就停產；沒想到二手價跌到百元美金後落入年輕人手中，成了嘻哈與電子樂的基石，1982 年的〈Planet Rock〉便是代表作。**這頁的大鼓用的正是同一套合成思路**——把一段被嫌棄的歷史，變成你指尖的玩具。
