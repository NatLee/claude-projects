# 真的假的？· 冷知識真偽鑑定所 🔴

> 每日小專案 · 2026-07-07 · 類別：網路趣聞・冷知識 · 形式：B 完整專案（Flask + SQLite）

網路上瘋傳的冷知識，有些是真材實料，有些卻是以假亂真的謠言。
這是一間「真偽鑑定所」——每抽一件案件，你要判斷這則敘述是 **真**、還是 **假（迷思）**，
蓋下印章後翻出正解、解說與**查證來源**，系統還會默默記錄你的戰績，
告訴你最容易被哪一類冷知識騙倒。

18 則題目「真・假各半」，全部經過查證。

---

## 怎麼玩

1. 讀一讀卡片上的敘述。
2. 蓋下你的判斷：**「屬實」** 或 **「謠言」**。
3. 朱印落下揭曉正解——**綠印＝真，朱印＝假**；上方橫幅告訴你這次鑑定對不對，下方附上白話解說與來源連結。
4. 按「下一件」繼續。累積連勝、答對率，挑戰把 18 件全部鑑定過。
5. 點開下方 **「我的鑑定檔案」**，看各類別答對率長條圖，以及「你最容易被◯◯類騙」的分析。

所有戰績存在本機的 `facts.db`（SQLite），關掉再開也還在；想從頭來過，按檔案裡的「清空紀錄」即可。

---

## 在 Windows 上啟動

你的電腦已裝 miniconda（Python），最簡單的方式是**雙擊 `run.bat`**：它會自動建立虛擬環境、安裝 Flask、啟動伺服器。

啟動後，用瀏覽器打開：

```
http://localhost:5000
```

要關閉伺服器，回到黑色視窗按 `Ctrl + C`。

### 手動啟動（conda 版）

若想自己來，打開 Anaconda Prompt：

```bat
cd /d "C:\Users\NatLee\Desktop\claude-projects\每日小專案\2026-07-07-真的假的"
conda create -n rensei python=3.11 -y
conda activate rensei
pip install -r requirements.txt
python app.py
```

一樣打開 `http://localhost:5000`。第一次執行時，`facts.db` 會自動建立並灌入題庫。

### API 一覽

| 方法 | 路徑 | 說明 |
|------|------|------|
| GET  | `/`             | 主頁面 |
| GET  | `/api/question` | 抽一題（優先給作答最少的、避開上一題），不含答案 |
| POST | `/api/answer`   | 送出判斷 `{fact_id, guess}`，回傳正解、解說、來源與最新統計 |
| GET  | `/api/stats`    | 整體統計、連勝、各類別答對率 |
| POST | `/api/reset`    | 清空作答紀錄（保留題庫） |

## 為什麼好玩

「判斷真假」比「單純讀冷知識」更黏人——因為你會忍不住想扳回一城。
而且這裡刻意混入了最愛在網路上流傳的**經典迷思**（10% 大腦、金魚 3 秒記憶、公牛看到紅色會抓狂、長城太空可見、維京角盔……），
一邊玩一邊幫你破除謠言。答錯不丟臉，被騙一次，就記住一輩子。
