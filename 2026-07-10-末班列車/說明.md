# 末班列車 · 一場沒有終點的夜行 🚆

> 深夜 23:47，你趕上了今天的最後一班車。車門合攏，廣播卻報出你從沒聽過的站名——
> 這班車，似乎不開往你以為的終點。

一款可以在自己電腦上跑的**互動文字冒險**（interactive fiction）。
一邊讀著溫柔又帶點奇幻的深夜列車故事，一邊做選擇，走向**六種不同的結局**。
你撿到的每一件小東西、每一段沒說出口的話，都可能悄悄改寫你的下車方式。

---

## 這是什麼？

- **類別**：創意・娛樂
- **形式**：完整小專案（Python Flask 後端 + SQLite 資料庫 + 前端單頁）
- **玩法核心**：23 個場景節點、多分支選擇、**物品／回憶條件解鎖**、**六結局收集**
- 你的旅程進度與「結局收集」都存在後端的 SQLite 裡，關掉瀏覽器再打開也還在。

### 六個結局（先不劇透，只給個影子）

| 結局 | 一句話 |
|------|--------|
| 歸途 | 最溫暖的一條路。 |
| 終點站 | 只為「還沒好好說再見的人」停車。 |
| 守燈人 | 有人得替最後一班車，一直點著燈。 |
| 貓站長 | 跟著一隻橘貓，會走到哪裡去？ |
| 霧見 | 沒帶燈的人，會留在霧裡。 |
| 天亮了 | 也許，這一切只是一場長長的夢……？ |

想全部收集齊，需要在不同的岔路做出不同的選擇——**提燈、銅鈴、一句沒說的再見**，
都是打開特定結局的鑰匙。

## 怎麼在 Windows 上啟動？

你的電腦已經有 miniconda（Python），兩種方式擇一即可。

### 方法 A：一鍵啟動（最簡單）

直接**雙擊資料夾裡的 `run.bat`**。
它會自動建立一個獨立的 Python 環境、安裝 Flask，然後發車。
看到畫面提示後，打開瀏覽器前往：

```
http://localhost:5000
```

要結束程式，回到黑色視窗按 `Ctrl + C` 即可。

### 方法 B：用 conda 手動啟動

打開「Anaconda Prompt」，依序輸入：

```bat
cd /d "%USERPROFILE%\Desktop\claude-projects\每日小專案\2026-07-10-末班列車"
conda create -y -n midnight python=3.11
conda activate midnight
pip install -r requirements.txt
python app.py
```

然後在瀏覽器打開 `http://localhost:5000`。

> 首次啟動時，程式會自動建立 `game.db`（SQLite 資料庫）並寫入整個故事，不需要任何額外設定。
> 想「洗掉所有進度與收集、從零開始」，把資料夾裡的 `game.db` 刪掉，再啟動一次就好。

## 檔案結構

```
2026-07-10-末班列車/
├── app.py              # Flask 後端：故事引擎 + API + SQLite 初始化
├── story_data.py       # 整個故事（23 個場景、選項、結局）的資料
├── templates/
│   └── index.html      # 前端單頁：深夜列車氛圍、打字機敘事、結局收集
├── requirements.txt    # 相依套件（只有 Flask）
├── run.bat             # Windows 一鍵啟動
└── 說明.md             # 就是這份
```

主要 API：`/api/state`（讀進度）、`/api/start`（重新上車）、
`/api/choose`（做選擇）、`/api/endings`（結局收集）。

## 為什麼好玩？

- **氛圍**：深色車窗、流過的暖黃燈火、一字一字浮現的敘事、濃霧與星河，
  讀起來像一部可以自己選路的短篇小說。
- **有重量的選擇**：接不接下車掌的提燈、要不要幫小孩保管銅鈴、
  願不願意為霧中的老奶奶停下腳步——不同的小決定，會通往截然不同的告別。
- **收集感**：六個結局像六張車票，全部集齊，你才看完這班車的全貌。

## 一個彩蛋：現實中真的有「幽靈車站」

故事裡的「霧見站」是虛構的，但**只在深夜、只為某些人存在的車站，現實中真的有**。
在遊戲的「結局收集 → 真實的幽靈車站」分頁裡可以讀到，這裡先講兩個：

- **紐約 · 市政廳站（City Hall）**：1904 年隨紐約第一條地鐵通車，是全網最華麗的一站，
  有拱形天窗、彩色玻璃磚與 Guastavino 的瓷磚穹頂。卻因月台太短容不下加長列車，
  **1945 年關閉**。至今仍有個小祕密：搭南向 6 號車坐到終點別急著下車，
  列車會沿環線繞過這座塵封的月台，讓你隔窗看它一眼。
- **倫敦 · 奧德維奇站（Aldwych）**：皮卡迪利線一條孤零零的支線，客流稀少，
  **1994 年停駛**。如今成了熱門拍片場景，《V 怪客》《最黑暗的時刻》等都在此取景。
