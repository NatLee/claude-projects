# 超級細菌 · 抗藥性當場演化給你看

## 這是什麼
把 2016 年那個著名的「MEGA-plate」實驗，變成你可以親手跑的演化模擬。抗生素從培養皿的兩端往中央一路加濃，分成幾道濃度牆，最高到 **1000 倍**致死劑量。細菌從無藥的邊緣出發、往中間長，每撞到一道更濃的藥牆就停住——直到邊緣某一隻**突變**出更強的抗藥性、突破過去，整片再跟著淹過去。看著看著，你就看懂了「天擇」是怎麼運作的。

## 怎麼玩
1. 按 **接種細菌**，兩端開始生長。
2. **速度**調快慢；**突變率**決定抗藥突變多久冒出一次——調高，你會看到細菌破牆的速度明顯加快（突變供給越多，演化越快）。
3. 看上方三個數字：天數、已演化到幾倍抗藥、佔領面積。
4. 當細菌攻下中央，代表它們已能在 **1000 倍致死劑量**下生長——超級細菌誕生。

## 為什麼會很酷（也很可怕）
- **演化不是抽象名詞，是看得見的**：細菌停在藥牆前的「猶豫」，就是在等一個夠幸運的突變；突變一出現，選擇壓力立刻把它推過牆。這正是抗生素濫用的縮影。
- **這是真實實驗**：哈佛 Kishony 團隊的 MEGA-plate（2×4 英尺）拍成的縮時影片震撼了全世界——大腸桿菌約 **11 天**就從無藥區演化到能扛 1000 倍藥的中央（Baym et al., *Science* 2016）。實驗還揭露一個反直覺現象：跑最快的不一定是抗藥性最強的；有時較強的突變被困在較快、較弱的族群後面。
- **和你我有關**：2019 年全球約 **127 萬**人死亡可直接歸因於抗藥性；若不改善，2050 年前恐累計超過 **3900 萬**人死亡。少用、正確用抗生素，就是在減緩這場演化。

## 運作原理（模擬怎麼算）
- 培養皿是一張細胞格網。每個空格屬於某道藥牆（需要的抗藥等級 0～4）。
- 細菌只能長進「自己抗藥等級 ≥ 該格藥牆」的格子；擴散到相鄰空格。
- 在藥牆前緣，後代有**小機率突變** +1 抗藥等級；一旦突變剛好達到牆的等級，就能破牆colonize，再往內擴散。這重現了「停頓→突變→突破」的階梯式征服。
- 以 node 無頭跑驗證：把突變率與速度調到最高時，細菌會真的突破到 1000× 並大面積佔領（模擬邏輯不會卡死或出錯）。
- 這是教學模型；真實抗藥性還牽涉水平基因轉移、抗藥代價、多種抗生素等，比這更複雜。

## 資料來源

- GBD/GRAM, *The Lancet*（2022、2024）全球抗菌藥物抗藥性負擔與 2050 年預測。

## 技術
純靜態單檔、零外部資源、離線可用。Canvas + `requestAnimationFrame`，離屏用小尺寸點陣圖放大繪製（`image-rendering:pixelated`）以求效能；分頁隱藏或離開視窗自動暫停，`prefers-reduced-motion` 時不自動播放、改由「繼續」手動推進。localStorage 只記你最快幾天攻下中央，key 加 `amr.` 前綴。
