# 每日小專案 📦

每天早上 6:00，這裡會自動長出一個全新的互動網頁小專案。
主題從四大類隨機抽選：網路趣聞・冷知識、生活痛點小工具、學習新知、創意・娛樂。

每個專案是一個獨立資料夾，裡面都有一份 `說明.md`。多數專案可直接用瀏覽器打開 `index.html`；少數是完整小程式（含 Python 後端），依各自的 `說明.md` 啟動。
