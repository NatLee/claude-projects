# 旋轉位置 · RoPE 旋轉位置編碼

把一個字的向量畫成**一排時鐘指針**，親手體會大型語言模型是怎麼「知道字的順序」的。

## 這是什麼

Transformer 的自注意力本身其實**看不見順序**——不加位置資訊的話，「貓 追 狗」和「狗 追 貓」在它眼裡是同一堆字。**RoPE（Rotary Position Embedding，旋轉位置編碼）** 的解法非常優雅：把每個字的 query/key 向量，依它在句子裡的**位置**旋轉一個角度。

高維向量可以兩兩一組拆成很多個 2D 平面，每一組就是一根「指針」。RoPE 在每個平面上把指針轉 `位置 × 該組頻率` 度，而且**每一組的頻率不同**：前面的像秒針飛快、後面的像時針幾乎不動。這頁把這件事整排畫出來，再親手驗證它最關鍵、也最反直覺的性質。

## 怎麼玩

1. **一排指針**：拖動「位置」滑桿，看 8 根不同頻率的指針一起轉動——快慢分明。按「自動巡航」讓它自己來回掃，秒針狂轉、時針微動的對比一目了然。
2. **相對距離**：放兩個字 A、B，各給一個位置 m、n。每個平面上會畫出兩根指針與它們之間的**夾角扇形**，右邊即時算出相似度（點積）。
   - 按 **「整句一起平移 ← / →」**：A、B 一起往同方向移，每個夾角**紋風不動**、相似度讀數一個字都不變（會亮出「🔒 紋風不動」）。
   - 打開 **「連動拖曳」**：拖任一條滑桿，整句一起動，效果相同。
   - 只動其中一條滑桿，讀數才會立刻改變——因為 **m−n 變了**。
   - 「換一組詞向量」可重擲一組示範向量。
3. **曲線**：把相對距離 `m−n` 從左掃到右，相似度畫成一條曲線；不管你把整句移到哪，只要 `m−n` 不變，你就永遠待在這條線上的**同一個金色點**。

無障礙：所有滑桿皆可用鍵盤操作、具聚焦樣式與 `aria-label`；相對距離與相似度會透過 `aria-live` 即時播報。系統若偏好「減少動態」，自動旋轉會停用，但拖動滑桿仍即時更新。

## 為什麼有趣：RoPE 的靈魂

核心是一條漂亮的數學等式。在每個 2D 平面上，把向量旋轉 θ 的矩陣是 R(θ)。RoPE 讓位置 m 的 q 轉 `m·ωᵢ`、位置 n 的 k 轉 `n·ωᵢ`。兩者的點積：

```
⟨R(m·ωᵢ)·qᵢ ,  R(n·ωᵢ)·kᵢ⟩  =  ⟨qᵢ ,  R((n−m)·ωᵢ)·kᵢ⟩
```

因為旋轉矩陣滿足 `R(α)ᵀ·R(β) = R(β−α)`。也就是說，旋轉後的點積**只取決於相對距離 (m−n)**，與兩個字落在句子的哪個絕對位置無關——卻又在計算過程中隱含了絕對位置。這帶來三個好處：

- **免額外參數**：RoPE 不需要學任何位置參數，純粹是一組固定的旋轉，直接套在既有的 Q、K 上。
- **天生相對**：模型自動獲得「相對位置感」，這正是語言最需要的（誰在誰旁邊、隔多遠）。
- **延伸超長**：關聯度隨相對距離自然衰減，能從短片語平滑外推到很長的上下文。

這頁的核心數學被抽成純函式，並用 Node 斷言驗證：

- `rotate2()` 的正確性與「保持長度」（等距變換）。
- **同時平移不變性**：對 20000 組隨機向量與位置，`dot(R(q,m),R(k,n))` 與 `dot(R(q,m+d),R(k,n+d))` 的最大偏差約 `1×10⁻¹³`（遠低於 1e-9 容差）。
- 點積確實**只是 (m−n) 的函數**（同樣 20000 組驗證）。
- 非退化檢查：只動其中一個位置，點積確實會變。

## 幾個誠實的說明

- 為了讓每根指針都清楚看得見、「快慢分明」，示範刻意**壓縮了頻率範圍**（面板 1 用 ω 從 1 到 1/64、面板 2 用 1 到 1/24）。真實模型的頻率是 `ωᵢ = 10000^(−2i/d)`，維度 d 常達數百，頻率橫跨上萬倍——最慢的那幾根在一整句裡幾乎不動，那正是它們用來編碼「大範圍位置」的方式。
- 面板 2、3 使用的是可重現的**隨機示範向量**（種子亂數），不是真實的詞向量；重點在於呈現「點積只認相對距離」這個對**任何**向量都成立的幾何性質。

## 技術與儲存

- 純靜態單一 HTML 檔，CSS 與 JS 全部內嵌；離線可用、雙擊 `file://` 即可開，零外部依賴（頁面上唯一的外部連結是指向 arXiv 論文的超連結）。
- 主視覺以 SVG 繪製；指針一律用純色描邊、旋轉走 `transform`，避免水平／垂直線在漸層描邊或逐路徑濾鏡下消失的常見地雷。
- 動畫只用 `transform`／`opacity`、以 `requestAnimationFrame` 節流，分頁隱藏或系統偏好「減少動態」時暫停自動旋轉。
- 使用 `localStorage` 記住上次的位置設定，key 皆加前綴 `rope.`：`rope.pos`、`rope.m`、`rope.n`。
