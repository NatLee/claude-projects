# 克拉德尼圖形 · 看得見的聲音

一塊撒了細沙的方形金屬板，在共振頻率上會讓沙子**逃離劇烈振動處、聚到完全不動的「節線」上**，排出對稱的幾何圖案。滑動頻率、切換模態，就能看見數千顆沙粒即時遷徙、重組成新圖形——**聲音變成看得見的形狀**。整頁為單一 HTML 檔、零外部資源、離線可直接開啟。

---

## 這是什麼

十八世紀的物理學家**恩斯特·克拉德尼（Ernst Chladni）**發現：在金屬板上撒細沙、用琴弓拉奏板緣使它共振，沙子會自動聚成清晰的線條圖案。這是人類第一次把「聲音的形狀」直接畫給眼睛看。本頁把這個古典實驗搬進瀏覽器：以物理模型即時模擬數千顆沙粒，讓你親手掃過一個個共振模態，看板上的幾何一張張翻新。

## 怎麼玩

- **頻率滑桿**：由低到高掃過 21 個共振模態（依克拉德尼定律的相對頻率排序）。每移動一格，沙粒就重新遷徙、聚成新的節線圖形。
- **精選圖樣**：九個預先算好節線縮圖的模態，點一下直接跳過去；含兩個「同相（＋）」邊界的特殊圖形。
- **邊界：反相 − / 同相 ＋**：切換兩種疊加符號，同一組 (m, n) 會給出兩種截然不同的圖案。
- **理論節線**：疊上一層青色細線，那是 `A(x, y) = 0` 的數學解——你會看到沙子恰好落在它上面，理論與模擬互相印證。
- **沙量 疏／中／密**：約 2200／3800／6000 顆，換檔即重新撒沙。
- **重新撒沙**：把沙倒回隨機位置，再看它們一次遷徙。
- **聲音（示意）**：**預設靜音**，點擊才啟動。以 Web Audio 純合成一個隨模態升高的正弦音，僅供聽覺對照，**不是板的真實頻率**。

## 為什麼有趣（哇時刻）

沙子不會思考，卻能排出對稱到不可思議的幾何。祕密在於：板一共振就形成**駐波**，振動最劇烈的「腹點」把沙彈開，完全靜止的「節線」留住沙。沙只是被動地被篩選、堆積，就親手描出了這塊板此刻看不見的振動模態。當你滑動頻率，成千上百顆沙粒同時起身、流動、在新的節線上重新落定——**你看見的是聲音的幾何**。

## 物理與數值模型

### 駐波振幅

方板駐波位移振幅採用經典**疊加近似**（x, y ∈ [0, 1]，邊長 L 正規化為 1）：

```
反相（−）：A(x, y) = cos(mπx)·cos(nπy) − cos(nπx)·cos(mπy)
同相（＋）：A(x, y) = cos(mπx)·cos(nπy) + cos(nπx)·cos(mπy)
```

其中 (m, n) 為模態指標。**節線**就是 `A(x, y) = 0` 的等值線；`|A|` 代表當地振動強弱，沙子會往 `|A|` 的極小處聚集。這是最常用來重現克拉德尼圖案的教學式近似；真實自由方板的模態需以 Rayleigh–Ritz 等方法求解、更為複雜，本頁取其形似而非精確解。頁面上的「理論節線」即以 **marching squares** 演算法在細網格上求出 `A = 0` 的等值線繪成。

### 沙的遷徙（模擬法）

撒 N 顆粒子（2200–6000），每一步對每顆粒子：

1. **帶雜訊的梯度下降**：在鄰近做數次隨機微擾探測，移動到其中 `|A|` 較小者（探測半徑正比於當地振幅——腹點探得遠、節線幾乎不動）。
2. **殘留抖動**：再加一個正比於 `|A|` 的隨機位移，另加極小的常數 shimmer 讓沙保持一絲生氣。

因為抖動幅度在節線上趨近於零，粒子一旦落到節線就幾乎凍結，於是自然堆積成清晰的線。切換 (m, n) 時粒子**保留位置**、就地朝新節線重新遷徙，這正是「圖形重組」的視覺來源。參數（`probeBase 0.004`、`probeGain 0.016`、`jitter 0.009`、`shimmer 0.0005`、探測數 2）經 Node 數值調校：約 350 步內、`|A| < 0.06` 的粒子佔比 ≥ 99%，收斂後 `|A|` 均值約 0.003–0.010，且沙散佈全板不塌縮。

### 克拉德尼定律與頻率

相對頻率以**克拉德尼定律** `f ∝ (m + 2n)²` 呈現：以最低模態 (1, 2) 的 `(1+4)² = 25` 為基準（×1.00），數值越大代表頻率越高、圖案越繁複。頻率滑桿即依此值由低到高排序各模態。示意音則採較和緩的單調映射並限幅於 110–1200 Hz，只為讓耳朵對照「模態越高、音越高」，非板的真實振動頻率。

> 註：克拉德尼定律原是對**邊界固定的圓板**歸納出來的（m 為直徑節線數、n 為圓形節線數）。本頁以它作為方板各模態的相對頻率示意，屬借用而非嚴格對應。

## 真實故事與查證

- **1787 · 首度公開**：德國物理學家兼音樂家**恩斯特·克拉德尼（Ernst Florens Friedrich Chladni，1756–1827）**在著作《**聲學理論的發現**》（*Entdeckungen über die Theorie des Klanges*）中公開此法：在金屬（或玻璃）板撒沙、以琴弓拉奏板緣使其共振，沙聚成節線圖形。他系統化地研究聲音與振動，被後世稱為「**聲學之父**」（也因對隕石的研究被稱為「隕石學之父」）。
- **原理**：板共振時形成駐波。**節點／節線**（位移為零、不動之處）聚集粉末；**腹點**（振動最大處）把粉末彈開。於是粉末描出該模態的節線分布。
- **克拉德尼定律**：對邊界固定的圓板，模態頻率 `f ∝ (m + 2n)²`，其中 m 為通過圓心的直徑節線數、n 為同心圓形節線數。頻率越高、圖形越繁複。
- **拿破崙與蘇菲·熱爾曼**：克拉德尼帶著實驗遊歷歐洲，**1809 年**在巴黎科學院當著**拿破崙**示範，拿破崙深受震撼，透過科學院設獎徵求能解釋這些振動圖形的**數學理論**。數學家**蘇菲·熱爾曼（Sophie Germain）**投入彈性板振動理論的研究，並於**1816 年**獲得該獎——這段插曲推動了彈性力學的發展。
- **迴響至今**：克拉德尼圖案至今仍是實用工具——**製琴師**用它替小提琴、吉他面板調音（觀察面板的模態以校正音色），工程上稱為**模態分析**；而把振動圖案當作視覺藝術的「**聲音成形**」（cymatics，Hans Jenny 於 1967 年推廣此詞）也是它的直系後代。

## 測試與驗證

以 Node 抽出頁面內嵌 `<script>` 並執行 `node --check`（語法通過），再 `require` 出其中的純函式做 **45 條斷言**，全部通過：

- **`amplitude` 已知點**：中心 (0.5, 0.5) 與各角點 (0, 0) 對「−」邊界為節點（值 = 0）、對「＋」邊界為腹點（值 = 2）；邊界已知節點 (0, 2/3) 對 (1, 2)− 為 0；非節點處 `|A| > 0.5`。
- **`|A| ≈ 0` 在節線上**：對多組模態用二分法沿掃描線求節點，`|A| < 1e-6`；再以 marching squares 線段中點交叉驗證 `|A| < 0.08`。
- **切換 (m, n) 改變零集合**：(0, 2/3) 是 (1, 2)− 的節點卻非 (2, 3)− 的節點；高模態節線段數大於低模態；同一 (m, n) 的 ± 邊界給出不同圖形。
- **頻譜與定律**：21 個模態依 `(m+2n)²` 遞增、皆滿足 m < n；`freqOf` 對照無誤；示意音頻率皆落在 110–1200 Hz。
- **模擬收斂**：撒 3000 顆、跑 350 步後，`|A| < 0.06` 的粒子佔比 ≥ 90%（實測近 100%），且沙散佈全板（16×16 網格覆蓋 ≥ 18 格，未塌縮成一點）。

## 技術備註

- **單一 HTML 檔**：CSS 與 JavaScript 全部內嵌，**零外部資源、零相依套件**，雙擊 `file://` 或離線皆可直接開啟；回首頁連結為相對路徑 `../../../index.html`。
- **繪製**：兩層 `<canvas>`——底層畫理論節線（僅在模態／開關變動時重畫），上層即時繪沙（以 `destination-out` 淡出做拖尾、`lighter` 加色堆疊，讓密處聚成發光的節線）。
- **不壞的 canvas 尺寸**：以 DPR 回寫 `width/height` 前先比對目標像素是否已相等才決定是否重設，避免 ResizeObserver→getBoundingClientRect×dpr 形成每幀翻倍的迴圈；畫布恆有 CSS 尺寸（容器 `aspect-ratio: 1/1`）。
- **省電**：`requestAnimationFrame` 迴圈在分頁隱藏（`visibilitychange`）、離屏（`IntersectionObserver`）或偏好減少動態時暫停。
- **prefers-reduced-motion**：完整降級——不做逐幀遷移，直接同步沉降並呈現最終沙圖；並以 `matchMedia` 的 `change` 事件動態監聽切換。
- **無障礙**：鍵盤完整可操作、互動元件有 `aria-label`／`aria-pressed`、狀態變化以 `aria-live` 即時播報、聚焦樣式清楚。
- **偏好儲存**：上次的模態、邊界、理論節線開關與沙量存於 `localStorage`，key 一律加 `chladni.` 前綴；聲音一律預設關閉、不記憶。
