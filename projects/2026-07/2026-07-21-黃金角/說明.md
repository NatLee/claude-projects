# 137.5° · 向日葵藏著的角度

一個把「黃金角」變成看得見的互動視覺化。撥動一個角度，看幾百顆種子從一團輪輻，瞬間收攏成向日葵那樣完美的雙向螺旋。

## 這是什麼

向日葵、松果、鳳梨、多肉植物的排列，背後常是同一條規則：**每長出一顆新種子（或一片新葉），就比前一顆多轉一個固定的「分歧角」**。當這個角度剛好是 **黃金角 ≈ 137.508°** 時，種子塞得最密、彼此最不擋光——這就是自然界反覆選中它的原因。

本頁用 **Vogel（1979）模型** 把這件事畫出來：

- 第 *i* 顆種子的極座標為 **θ = i × 分歧角**，**r = c × √i**。
- √i 的間距讓「每顆種子分到的面積」大致相等，於是整片密度均勻。
- 種子在中心生成、往外遷移，正如真花。

## 怎麼玩

- **拖動「角度」滑桿**：即時看圖樣變形。
- **按「✦ 對齊黃金角」**：角度平滑掃到 137.5°，輪輻收攏成向日葵，右上角亮起金色徽章。
- **預設角度**：`90°`（塌成放射狀輪輻）、`120°`、`137.3°`（差一點點，螺旋鬆開成緩緩繞行的手臂）、`150°`。
- **種子數**：150–1400 顆。
- **顯示螺旋臂**：疊上 34 / 55 / 89 條螺旋。做法是把第 *i* 顆與第 *i+F* 顆種子連線，恰好描出 **F 條**螺旋——**在黃金角時線條乾淨漂亮，離開黃金角就纏成一團**，直觀顯示黃金角的特別。

## 為什麼有趣

1. **一個角度定江山**：花不會數數，卻總排出斐波那契數目的螺旋（34、55、89、144…，都是相鄰的斐波那契數）。這不是花在數，而是黃金角的必然幾何結果。
2. **黃金角是一條刀鋒**：φ =(1+√5)/2 是「最難用分數逼近」的無理數，因此黃金角讓每顆新種子都落進前面所有種子留下的最大縫隙。只要偏離 0.1–0.2°，螺旋就會鬆脫、露出縫隙、種子開始互相擋光。頁面上撥到 137.3° 就看得到。
3. **看得見的數學**：Vogel 的兩行公式，重現了整朵向日葵的視覺複雜度。

## 黃金角怎麼算

黃金角 = 360° × (2 − φ) = 360° / φ² ≈ **137.50776°**，其中 φ =(1+√5)/2 ≈ 1.6180339。

## 技術

- 純靜態單檔頁面，Canvas 2D 繪製，零外部相依（連 CDN 都不用），雙擊 `file://` 即可開。
- 核心座標計算抽成純函式 `seedPositions(n, angleDeg, maxR)`，以 node 斷言驗證（中心種子在原點、r 隨 √i 單調遞增、最外種子貼齊 maxR、角度步進正確）。
- 動畫只用 canvas 重繪；生長動畫一次性 ≤1.1 秒，分頁隱藏或 `prefers-reduced-motion` 時直接呈現最終狀態並停止 rAF。
- Canvas 尺寸由 CSS 決定（`aspect-ratio:1` + 100% 高），JS 只讀取尺寸 ×dpr 回寫繪圖緩衝，不回寫 CSS，避免每幀翻倍的回饋迴圈。
- **localStorage**：以 `phyllo.` 前綴儲存最後一次的角度、種子數與螺旋設定（key：`phyllo.state`），重開頁面會沿用。

- Helmut Vogel, *"A better way to construct the sunflower head"*, Mathematical Biosciences 44 (1979) — Vogel 模型 θ=i·137.5°, r=c√i。
- Wikipedia：Golden angle、Phyllotaxis、Fibonacci number（斐波那契螺旋數、黃金角 ≈137.508°、φ 最難逼近的性質）。
