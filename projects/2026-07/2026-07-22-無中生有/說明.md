# 無中生有 · 從一縷能量湊出整個宇宙

一款純靜態的「合成 × 收集」小遊戲：從最原始的三種元素（能量、空間、時間）出發，兩兩相湊，一路合成出恆星、海洋、生命、恐龍、人類，最後湊出「文明」與一個小小的彩蛋。目標是填滿整張 36 格的宇宙圖鑑。

## 怎麼玩

1. 右邊是「宇宙圖鑑」，一開始只有三格是亮的（能量 ⚡、空間 🌌、時間 ⏳），其餘都是 `？`。
2. 點一個亮起的元素放進左邊工作台的第一格，再點一個放進第二格，就會自動合成。
   - 相同的兩個也可以：例如「能量＋能量＝粒子」。
   - 點工作台裡的格子可以把它取出來。
3. 湊出新元素時，圖鑑對應的 `？` 會翻出來，卡片上附一句查證過的小知識。
4. 卡住時按 **💡 提示**，它會找出一對「湊得出新東西」的元素、讓它們發亮。
5. **↺** 可以重新開始（會清空進度）。

進度會存在瀏覽器裡（localStorage key 前綴 `cosmos.`），下次打開接著玩。

## 合成路線（劇透）

整張圖鑑刻意照著「宇宙演化史」排列，大致是：

- **物理**：能量 → 粒子 → 原子 → 恆星 → 超新星 → 重元素 → 星塵 → 行星
- **化學**：原子＋重元素 → 水 → 海洋 → 有機分子
- **生物**：有機分子 → 生命 → 植物 → 氧氣 → 動物 → 大腦 → 人類
- **文明**：語言 → 文明 → 文字 → 電 → 電腦 → 網路 → AI → 🎁

另有幾個支線收集品：星系、月亮、雲、彩虹、恐龍、好奇心。

一共 36 個元素、33 條配方。整張配方圖經程式驗證：從起始三元素一定湊得出全部、每一對原料只對到一個產物、沒有任何死路（見「測試」）。

## 為什麼好玩

- **從無到有的爽感**：每一步都對得上真實的宇宙演化順序，湊著湊著就把 138 億年走了一遍。
- **收集慾**：圖鑑上的 `？` 一格格被點亮，看得見進度。
- **偷渡知識**：每張卡片一句話小知識，玩完順便記住了一串宇宙冷知識。

## 卡片小知識的資料來源

卡片上的數字都查證過，主要依據：

- 大霹靂約 138 億年前、複合（原子形成）約在 38 萬年、第一批恆星約 1 億年 — NASA Science（Webb / Early Universe）
- 重元素在恆星與超新星中鍛造（stellar nucleosynthesis、"star stuff"） — Carl Sagan；astronomy.com、science.org
- 地球約 45.4 億年、最早生命證據約 35–38 億年前 — NASA Astrobiology、Wikipedia
- 智人約 30 萬年前（Jebel Irhoud） — Smithsonian Human Origins
- 大氧化事件約 24 億年前、恐龍稱霸逾 1.6 億年 / 6600 萬年前滅絕、米勒實驗 1953 — Wikipedia

## 技術

- 單一 `index.html`，純 HTML / CSS / JS，無外部相依、無後端，雙擊 `file://` 即可玩。
- 動畫只用 `transform` / `opacity`；火花用短命的 CSS 動畫、播完即移除，沒有常駐 rAF 迴圈或 canvas。
- 完整支援鍵盤操作、`aria-live` 播報結果，並對 `prefers-reduced-motion` 降級（含動態監聽）。
- 元素與配方集中在 HTML 的 `COSMOS_DATA` 區塊，是唯一資料來源。

## 測試

配方圖的正確性用純資料驗證（可解性、唯一性、無死路、finale 最後才出現），驗證腳本會直接從本頁 HTML 抽出 `COSMOS_DATA` 區塊執行，確保頁面內的資料與測試一致。全站另跑 `node tools/check.js` 通過。
